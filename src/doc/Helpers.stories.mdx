# Helpers

## Class utils

_Basscss_ is embedded by default to offer simple class utils.
Check [the documentation](https://basscss.com/) for more informations.

## Global css vars

Here is the list of global css variables with their default values,
that you can use or alter to style components your way.

```
--snz-primary: #343434;
--snz-snackbar-bg: #343434;
--snz-icon-primary: #343434;
--c-button-unactive: #343434;
--snz-bg-white: #ffffff;
--snz-bg-low: rgba(0, 0, 0, 0.05);
--snz-bg-dark: rgba(0, 0, 0, 0.2);

// Icon colors
--snz-icon-active: #5f6368;
--snz-icon-hover: #7f7f7f;
--snz-icon-click: #f2f2f2;
--snz-icon-bg-default: #ffffff;
--snz-icon-bg-hover: #f2f2f2;
--snz-icon-unactive: #c4c4c4;

// Font colors
--snz-font-high: rgba(0, 0, 0, 0.8);
--snz-font-medium: rgba(0, 0, 0, 0.6);
--snz-font-low: rgba(0, 0, 0, 0.5);
--snz-font-disabled: rgba(0, 0, 0, 0.3);
--snz-font-inverted: #ffffff;
```
